# GIS Data Type Classifier

This beginner Python project automatically classifies GIS-related file types.

## Features
- Detects vector files (.shp)
- Detects raster files (.tif)
- Detects table files (.csv)

## Skills Used
- Python loops
- if-elif conditions
- String slicing
- Automation logic
